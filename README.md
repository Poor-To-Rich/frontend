## 📖 프로젝트 소개

### 부자될거지(Poor-To-Rich) - 스마트 가계부 서비스

> 수입과 지출을 한눈에 관리할 수 있는 가계부 <br/>
> 월별·연도별 소비 및 수입 패턴을 차트로 분석하여 편리하게 확인 <br/>
> "지갑 봉합소" 채팅방에서 다양한 사람들과 절약 노하우를 실시간으로 공유 <br/>

##### 효율적인 소비 관리와 함께, 절약의 재미까지 경험해 보세요! 💰
